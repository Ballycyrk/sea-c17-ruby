## Array Guidelines 

#### No spaces before `[`

```ruby
# bad
some_array [42]

# good
some_array[42]
```


#### No spaces after `[` or before `]`

```ruby
# bad
some_array[ 42 ]

# good
some_array[42]
```


#### Prefer `size` over `length` and `count`

```ruby
# bad
some_array.length
some_array.count

# good
some_array.size
```
